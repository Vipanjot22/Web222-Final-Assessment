1. Explain the concept of Responsive Design. What does the term mean?
Ans. Responsive design is a visual user interface (GUI) design strategy for creating content that adapts to different screen sizes without losing its quality. Designers use relative units (percentages) and media queries to ensure that their designs adjust to the browser space and maintain content uniformity across devices. Responsive design is a somewhat retro approach to web site design that addresses many of the design issues that have arisen as a result of the growth of new sorts of mobile devices. Instead of fixed-width settings, responsive design pages employ X and Y coordinates on a grid for layout and mathematical percentages for pictures. Using percentages instead of fixed-width parameters with a grid layout results in a more fluid layout that adapts to the size of the display.

2. How does responsive design relate to web technologies?
Ans. We're soon approaching the point where we won't be able to keep up with the never-ending new resolutions and devices in the world of Web design and development. Creating a website version for each resolution and each device would be unfeasible, or at the very least impracticable, for many websites. To resolve this, we use idea of responsive web design in many ways. For example, there are many responsive web design tools available to design and customise websites. One of them is Bootstrap. It is the most popular framework to build a responsive and mobile first websites. It is basically a collection of usable code which is written in html, CSS and JavaScript. Moreover, Bootstrap comes up with its own grid system which is predefined which we can use easily. Also, Cascading Style Sheets (CSS) are used in responsive web design to offer varied style attributes based on the screen size, orientation, resolution, colour capabilities, and other aspects of the user's device. The viewport and media queries are two examples of CSS attributes related to responsive web design. responsive Web design is not only about adjustable screen resolutions and automatically resizable images, but rather about a whole new way of thinking about design.
